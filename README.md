# 📊 Счетчик букв и цифр в тексте

Данная программа написана на Python и предназначена для подсчета количества букв, цифр и прочих символов во введенном пользователем тексте.

🚀 Описание работы

После запуска программа предлагает пользователю ввести любой текст. Затем она анализирует каждый символ и определяет, к какой категории он относится:

Буквы — учитываются как латинские, так и кириллические символы (включая ё и Ё)
Цифры — от 0 до 9
Прочие символы — знаки препинания, пробелы, спецсимволы и т.д.

В конце программа выводит:

количество цифр
количество букв
количество остальных символов

#1 🧠 Как это работает

Программа проходит по каждому символу строки и проверяет, есть ли символ в списке букв
Если нет — проверяет, является ли он цифрой
Если не подходит ни под одну категорию — относит его к прочим символам
▶️ Пример использования

Введите ваш текст: Hello 123!

Колличество цифр в вашем тексте: 3

Колличество букв в вашем тексте: 5

Колличество символов, не отнощясимся к буквам и цифрам в вашем тексте: 2
